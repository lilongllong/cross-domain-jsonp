# cross-domain-jsonp
分别使用js和jquery实现 jsonp跨域   
服务器使用express创建
## 文件目录
> jsonp-js为使用原生js 创建script标签实现跨域

> jsonp-query为使用jQuery,创建Ajax和getJSON方法分别实现jsonp跨域请求

## clone
```
git clone https://github.com/lilongllong/cross-domain-jsonp.git
```

## 安装
两个文件分别为各自的git目录
在各自目录下执行
```
npm init
npm install --save-dev express

```

## 运行
```
npm start
```
